- SmartDNS 是一个运行在本地的 DNS 服务器，它接受来自本地客户端的 DNS 查询请求，然后从多个上游 DNS 服务器获取 DNS 查询结果，并将访问速度最快的结果返回给客户端，以此提高网络访问速度。 SmartDNS 同时支持指定特定域名 IP 地址，并高性匹配，可达到过滤广告的效果; 支持DOT(DNS over TLS)和DOH(DNS over HTTPS)，更好的保护隐私。
    
    

这个是官方上写的一段话, 我打算使用SmartDNS的原因, 第一是因为Clash的DNS解析多多少少有点慢, 第二是这个DNS的配置看起来还算比较容易上手.

唯一感到有美中不足的是没有统计接口, 看不到各个DNS的一个数据情况.

```
# Add custom settings here.

# set log level
# log-level [level], level=fatal, error, warn, notice, info, debug
log-level info # 日志等级

# log-size k,m,g
log-size 1m  # 日志大小

# log-file /var/log/smartdns.log
# log-num 2

# List of hosts that supply bogus NX domain results
# bogus-nxdomain [ip/subnet] # 被污染的IP

cache-size 512 # 缓存大小, 条目

speed-check-mode tcp:80,tcp:443,ping # 测速手段, 最优先的是80端口的TCP测速

# -group domestic组的DNS
# 同时也是默认使用的DNS
server 192.168.1.1 -group domestic
server 202.101.172.35 -group domestic
server-tls 1.12.12.12:853 -group domestic
server-tls 120.53.53.53:853 -group domestic
server-tls 223.5.5.5:853 -group domestic
server-tls 223.6.6.6:853 -group domestic
server 114.114.114.114 -group domestic
server 114.114.115.115 -group domestic
server 1.2.4.8 -group domestic
server 210.2.4.8 -group domestic

# -group fq_dns 第二组DNS
# ----- Default Group -----
# 默认使用的上游 DNS 组
# OpenDNS 非常规 443 端口、支持 TCP 查询
server-tcp -group fq_dns 208.67.220.220:443
# OpenDNS 的 IP DoH
server-https https://146.112.41.2/dns-query -group fq_dns
# TWNIC 的 IP DoH
server-https https://101.101.101.101/dns-query -group fq_dns
# 你也可以配置其它 DNS 作为上游
server-tls 1.1.1.1:853 -group fq_dns
server-https https://jp.tiar.app/dns-query -group fq_dns
server-https https://jp.tiarap.org/dns-query -group fq_dns
```
